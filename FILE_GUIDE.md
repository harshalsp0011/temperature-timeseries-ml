## üéØ How to Use This Guide

### For New Team Members
1. Start with **QUICK_START.md**
2. Read the main **README.md**
3. Review **04_Reports/Final_Reports/** for conclusions
4. Check **05_Research_Papers/** for background

### For Continuing Analysis
1. Open **01_Source_Code/Python/PyCode_Stat.ipynb** or **01_Source_Code/R_Code/Team12Code.Rmd**
2. Use **02_Data/Raw/US_City_Temp_Data.csv** as input
3. Output processed data to **02_Data/Processed/**

### For Understanding the Project
1. **04_Reports/Final_Reports/final report Time Series Analysis.docx** - Complete analysis
2. **05_Research_Papers/** - Theoretical foundation
3. **01_Source_Code/** - Implementation details

### For Contributing
1. Update code in **01_Source_Code/**
2. Save processed data in **02_Data/Processed/**
3. Update documentation in **03_Documentation/**
4. Add findings to **04_Reports/**
